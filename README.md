# VYANTARA

## AI-Powered Occlusion-Robust Road Extraction and Graph-Theoretic Criticality Analysis

VYANTARA is an AI-powered geospatial intelligence platform developed for intelligent urban road analysis from satellite imagery.

## Features

- SegFormer-based Road Extraction
- Skeletonization
- Graph Construction
- Graph Healing
- Critical Junction Detection
- Disaster Simulation
- Road Network Resilience Analysis
- Interactive Streamlit Dashboard

## Technology Stack

- Python
- PyTorch
- Hugging Face Transformers
- NetworkX
- OpenCV
- Streamlit

## Project Workflow

Satellite Image
→ Road Extraction
→ Skeletonization
→ Graph Construction
→ Graph Healing
→ Criticality Analysis
→ Disaster Simulation
→ Resilience Analysis
→ Dashboard

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
